# PasswordSecurity-JohnTheRipper
Educational project on password security using John the Ripper. Demonstrates hash cracking techniques, search space analysis, and performance evaluation using mask attacks on sample MD5 hashes in a controlled academic environment

This project was developed as part of a university course on **Information Systems Security**. It focuses on password authentication security and demonstrates how weak passwords can be identified using hash cracking techniques.

## ⚠️ Disclaimer
This project is strictly for **educational purposes**.  
All hashes used are **synthetic/test data** and do not correspond to real user credentials.

---

## 📌 Project Overview

The goal of this project is to:
- Understand how password hashing works
- Analyze password strength
- Evaluate cracking techniques using **John the Ripper (JtR)**
